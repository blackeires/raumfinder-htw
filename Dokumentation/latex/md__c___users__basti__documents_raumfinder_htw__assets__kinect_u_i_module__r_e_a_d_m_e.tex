Das Kinect\+UIModule beinhaltet die Gestensteuerung und Gestenerkennung. In erster Linie werden hier die Gesten und Hand -\/ Zustände prozessiert.

Das Modul beinhaltet\+:
\begin{DoxyItemize}
\item Die verschiedenen Klassen für die Cursos (normaler Cursor and Waiting Cursor)
\item Das Input\+Modul, um die eingehende Daten der Kinect zu verarbeiten und die damit resultierenden Aktion durchzuführen
\item Die Beschreibung der Kinect Daten, welche in das Modul geladen werden
\item Das Modul ist angewiesen auf das Unity Pro Package von Microsoft, welches bereits fertigen Tracking Code liefert und sich im {\ttfamily Kinect} Order befindet
